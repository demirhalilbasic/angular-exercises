Data Binding - Demonstracija

1. Interpolation (Komponenta → Template)

Prikaz varijabli u HTML-u

{{ naslov }}

Trenutno vrijeme: {{ trenutnoVrijeme }}

{{ '{{ naslov }}' }}

2. Property Binding (Komponenta → Template)

Postavljanje HTML svojstava

[src]="slikaUrl"
[disabled]="isDisabled"

3. Event Binding (Template → Komponenta)

Reagovanje na korisničke akcije

{{ poruka }}

{{ brojKlikova }} klikova
(click)="klikniMe()"

4. Two-Way Binding (Komponenta ↔ Template)

Sinhronizacija podataka

Unijeli ste: {{ korisnickoIme || '(prazan unos)' }}

Email: {{ email || '(prazan unos)' }}

Komentar ({{ komentar.length }} karaktera):

{{ komentar || '(prazan unos)' }}

[(ngModel)]="korisnickoIme"

5. Kombinacija svih vrsta Data Binding-a

Interaktivna forma

Sažetak unesenih podataka:

  • Korisničko ime: {{ korisnickoIme || 'nije uneseno' }}
  • Email: {{ email || 'nije uneseno' }}
  • Broj klikova: {{ brojKlikova }}
  • Komentar: {{ komentar || 'nije uneseno' }}
